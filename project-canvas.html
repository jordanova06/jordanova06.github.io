Blank Slate

Intro

As someone who comes from an artistic background, I’ve always wondered what it took to make the applications that I’ve used often. I decided I wanted to learn for myself so I pushed myself and ventured off in uncharted territory for me, which is HTML Canvas.

Approach

Since it was something I haven’t worked with before I had to do extensive research on how HTML Canvas works, starting from basic shapes to free form drawing. I looked through all of the functionalities that Canvas offers and decided I wanted to make a simple drawing app, one that works similarly to MS Paint.

The process

After all the preparations, setting up the position of the canvas and the toolbars was my first step. Secondly, I had to make each tool have its own attributes so that it gave the feel of using a different marker, pencil, brush etc. Afterwards, I worked on the available colors that the app offered and the ability to choose a custom color of your own. Lastly, I gave the user the ability to undo a certain action or clear the entire canvas and start drawing anew. The hardest part of the project was trying to understand the algorithms that are used in drawing shapes and color filling, but watching it all come together in the end was worth it.

Ending notes

This was definitely a project that pushed the limits of my JavaScript knowledge but I’m very glad that it’s a challenge I took on and it resulted in a project that I’m very proud of. My plan is to further develop this app in the future so that I can make it responsive for a pen tablet and add in more useful functionalities.
